Como usar

Este datepicker combina a simplicidade da Petite-Vue com a funcionalidade completa do Flatpickr. Basta clicar em qualquer campo de data para abrir o seletor.

// Inicialização básica
<input type="text"
  @vue:mounted="initFlatpickr($el, {})">

// Com opções personalizadas
<input type="text"
  @vue:mounted="initFlatpickr($el, {
    enableTime: true,
    mode: 'range',
    minDate: '2023-01-01'
  })">
